# 🚀 MediaPipe AI Studio

A collection of real-time Computer Vision projects built using **MediaPipe**, **OpenCV**, and **Python**.

This repository combines multiple MediaPipe applications into one project, making it easy to explore different Computer Vision solutions from a single place.

---

## 📌 Features

- 😀 Face Detection
- 🎭 Face Mesh
- ✋ Hand Tracking
- 🧍 Pose Estimation
- 🤖 Holistic Tracking
- 📦 Object Detection
- 📐 3D Object Detection (Objectron)
- 🚶 Motion Tracking
- 👤 3D Face Transformation
- 🏃 Body Pose Tracking
- 📷 Webcam & Video Support

---

## 🛠️ Technologies Used

- Python
- MediaPipe
- OpenCV
- NumPy
- Matplotlib

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/MediaPipe-AI-Studio.git
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the project

```bash
python app.py
```
